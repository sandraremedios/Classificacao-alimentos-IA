# Classificação de Sentimentos em Avaliações de Produtos Alimentícios utilizando Machine Learning

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como trabalho final da disciplina de Inteligência Artificial da Universidade Federal do Oeste do Pará (UFOPA).

O objetivo é desenvolver um modelo de Machine Learning capaz de classificar automaticamente avaliações de produtos alimentícios em sentimentos positivos ou negativos utilizando técnicas de Processamento de Linguagem Natural (PLN).

---

## 🎯 Objetivo

Criar um modelo de classificação de sentimentos utilizando a biblioteca Scikit-Learn e o algoritmo Multinomial Naive Bayes para identificar automaticamente avaliações positivas e negativas.

---

## 🧠 Técnica de IA Utilizada

- Machine Learning Supervisionado
- Vetorização de texto com TF-IDF
- Algoritmo Multinomial Naive Bayes

---

## 📊 Base de Dados

Foi utilizado o conjunto de dados **Amazon Fine Food Reviews**, contendo avaliações de consumidores sobre produtos alimentícios.

Fonte:

https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews

## ▶ Como Executar

1. Abra o notebook `Projeto_IA.ipynb` no Google Colab.
2. Faça o upload do arquivo `Reviews.csv`.
3. Execute todas as células do notebook.
4. O modelo será treinado automaticamente.
5. Ao final serão exibidas as métricas de desempenho e a matriz de confusão.
